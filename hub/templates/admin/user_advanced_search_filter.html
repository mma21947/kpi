Advanced query search

  • You can use django field lookups to perform advanced searches
  • Some examples:
  • logged in after March 1, 2023: 
    last_login__gte:2023-03-01
  • emails not from domain.tld: 
    NOT email__iendswith:@domain.tld
  • Users being pending removed: 
    extra_details__date_removal_request__isnull:False
    AND extra_details__date_removed__isnull:True
  • Users removed: 
    extra_details__date_removed__isnull:False
